--- title: Supprimer les pointillés entourant vos liens sur Firefox layout: default tags: - css type: regular ---

Si vous détestez autant que moi ces petits pointillés qui s'affichent sur Firefox, vous êtes chanceux, voici la solution. Il s'agit d'un contour défini par la propriété CSS outline (contour en anglais bien sûr). Il existe a ma connaissance deux méthodes très simple à utiliser, la première consiste tout simplement à désactiver la propriété CSS outline sur toute votre page (vous pourrez toujours la redéfinir au besoin) : 

a {
  outline: none;
}

Il existe une autre solution qui s'applique uniquement à Firefox, la propriété -moz-outline-style mais celle-ci n'est pas valide :

:focus {
  -moz-outline-style: none;
}

J'ai pris l'habitude d'utiliser la propriété outline plutôt que
-moz-outline-style et je l'ajoute toujours dans mon fichier CSS tout comme le border : none sur la balise <img>.
