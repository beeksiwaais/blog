Github est devenu la plateforme en ligne à la mode pour tout ce qui concerne le "versionning" de projet.
Le côté réseau social y est sûrement pour grand chose mais pas seulement, il est extrêmement simple des suivre et de publier des "issues" et on est toujours content de trouver un projet bien documenté avec un Wiki. Cette fonctionnalité est disponible pour tous les projets et il suffit de créer une branche dédié à cette tâche. Si vous trouviez cela encore trop compliqué, Github a décidé de nous simplifier la vie avec Gollum.

Gollum est utilisé par Github pour gérer les wikis sur le site mais vous pouvez également vous en servir sur votre machine. Pour cela rien de plus simple $ gem install gollum puis à la racine de votre projet (qui utilise Git bien entendu mais pas obligatoirement avec Github) $ gollum. Voilà un serveur web est lancé sur un port 4000 et quelques; il ne vous reste plus qu'à éditer ou à consulter votre wiki.
